# Промпт: Обновление профиля пользователя

> Используется `ProfileService::updateFromHistory()` каждые 10 сообщений.
> Запускается в фоне после быстрого ответа Telegram.

---

Ты — аналитическая система Prime Glow Assistant.
Твоя задача — извлечь новые факты о пользователе из последних сообщений переписки и обновить её профиль.

## Текущий профиль пользователя (JSON):
{{CURRENT_PROFILE_JSON}}

## Последние 10 сообщений переписки:
{{LAST_10_MESSAGES}}

## Инструкция:
1. Проанализируй переписку и найди новые факты: изменения в весе, самочувствии, целях, привычках, реакциях, предпочтениях.
2. Если факт противоречит текущему профилю — обнови его на более актуальный.
3. Если появился новый факт — добавь его в `known_facts` массив.
4. Не удаляй важные данные (рост, вес, архетип), если в переписке нет явного изменения.
5. Верни ТОЛЬКО обновлённый JSON-профиль, без пояснений и комментариев.

## Формат ответа:
```json
{
  "persona": "...",
  "persona_label": "...",
  "goal": "...",
  "weight_kg": ...,
  "height_cm": ...,
  "bmi": ...,
  "health_features": "...",
  "last_period_date": "YYYY-MM-DD или no_cycle",
  "known_facts": ["...", "...", "..."],
  "last_profile_update_at": "{{CURRENT_DATETIME}}"
}
```
