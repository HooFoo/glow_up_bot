# Пользовательский поток бота: Тексты сообщений

Этот файл содержит все тексты, которые бот отправляет по шагам онбординга.
`MessageService` читает нужный текст по ключу и отправляет через Telegram API.

---

## START_WELCOME
> Триггер: команда `/start`, пользователь новый

```
Привет 🤍
Я — твой Prime Beauty AI-ассистент.

Я помогу тебе собрать систему: питание, цикл, антистресс, ритуалы, уход — без хаоса и насилия над собой.

Важно: я не врач и не заменяю медицинские рекомендации. Я — инструмент осознанности и структуры.

Готова настроить свой Prime-профиль?
```

> Кнопка: `[НАЧАТЬ НАСТРОЙКУ]`
